# Drug Advisory System Using AI

## Description
The Drug Advisory System using AI is designed to assist users by providing drug recommendations,
usage guidelines, and precaution information based on user inputs such as symptoms and medical data.
The system uses artificial intelligence techniques to improve accuracy and efficiency.

## Technologies Used
- Python
- Machine Learning
- AI Algorithms
- Flask (if used)
- Dataset for drug information

## Features
- Suggests appropriate drugs based on symptoms
- Provides dosage and usage guidelines
- Displays precautions and side effects
- User-friendly interface

## How to Run the Project
1. Install Python on your system
2. Download or clone this repository
3. Install required libraries using:
   pip install -r requirements.txt
4. Run the main file:
   python app.py
